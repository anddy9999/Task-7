Task 7: Identify and Remove Suspicious Browser Extensions

SUSPICIOUS EXTENSIONS REMOVE REPORT

1.	Executive Summary :

This report documents the process of identifying, analyzing, and removing potentially suspicious browser extensions. The objective was to enhance browser security awareness and understand the risks associated with malicious or unnecessary extensions.


2. Objective :
•	Learn how to identify harmful browser extensions.
•	Evaluate extension permissions and associated risks.
•	Remove suspicious or unnecessary extensions.
•	Improve overall browser security posture.

3. Tools Used :
•	Google Chrome 
•	Browser Extension Manager
•	Online research for extension reviews and risk analysis

4. Methodology :

The following structured approach was used: 
    1. Accessed the browser extension manager.
2. Reviewed all installed extensions carefully. 
3. Analyzed permissions such as website data access and browsing history access.
4. Evaluated risk based on developer credibility and user reviews.
5. Identified unused or suspicious extensions.
6. Removed unnecessary extensions and restarted the browser.

5. Findings :

6. Security Risks of Malicious Extensions :
•	Data theft (credentials, cookies).
•	Session hijacking.
•	Keystroke logging.
•	Adware injection.
•	Browser performance degradation.
•	Privacy violations.


7. Preventive Recommendations :
•	Install extensions only from trusted sources
•	Review permissions before installation
•	Remove unused extensions regularly
•	Keep browser updated
•	Enable multi-factor authentication (MFA)
•	Use enterprise browser policies (in corporate environments)

8. Internship Learning Outcomes :

•	Understood browser-based attack vectors.
•	Learned to evaluate extension permissions critically.
•	Improved awareness of privacy and data risks.
•	Enhanced practical cybersecurity auditing skills.
•	Developed professional documentation skills.


9. Conclusion :

This task enhanced practical understanding of browser security management. By identifying and removing suspicious extensions, the overall attack surface was reduced. Regular auditing of browser extensions is an essential cybersecurity best practice.
